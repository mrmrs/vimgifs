Use :g/^/m 0 to move every line in the file to the start of the file (0). In other words this command reverses the entire file.

The next example command uses a range of lines to reverse :5,15g/^/m 0 And lastly a visual selection can be reverse sorted as well.

g means global
m means move
^ is regex for beginning of line
